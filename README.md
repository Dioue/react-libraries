# Frontend Libraries Projects

This folder contains several projects from freeCodeCamp's JavaScript Algorithms and Data Structures curriculum. These projects help practice  React.js, HTML, and CSS development without using React or other frameworks.

## Projects Overview

### 25+5 Clock
- **Scope**: Digital clock with 24-hour/12-hour format toggle, alarm functionality, and customizable background/foreground colors.
- **Key Concepts**: Date manipulation, event handling, DOM updates

### Drum Machine
- **Scope**: Virtual drum machine with keyboard and button controls, sound playback, and visual feedback.
- **Key Concepts**: Audio API, event listeners, CSS animations

### JavaScript Calculator
- **Scope**: Basic arithmetic calculator with memory functions and error handling.
- **Key Concepts**: DOM manipulation, expression parsing, error handling

### Markdown Previewer
- **Scope**: Live preview tool for Markdown syntax with customizable themes.
- **Key Concepts**: Text processing, live DOM updates, CSS theming

### Quote Generator
- **Scope**: Displays random quotes from API with sharing functionality.
- **Key Concepts**: API integration, DOM manipulation, CSS transitions

## Limitations
- No persistent storage or user authentication
- Basic styling only (no responsive design)
